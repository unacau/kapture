Kapture Window Events Test

This page demonstrates connecting to Kapture via window events.

Make sure the Kapture extension is installed and check the console for logs.

Waiting for auto-connect...

How to use:

  1. Open the browser console (F12)
  2. Click "Connect to Kapture" to establish connection
  3. Check the extension popup/badge to see the connection status
  4. Click "Disconnect from Kapture" to close the connection

Code Example:

// Connect to Kapture
window.dispatchEvent(new CustomEvent('kapture-message', {
    detail: { type: 'connect' }
}));

// Disconnect from Kapture
window.dispatchEvent(new CustomEvent('kapture-message', {
    detail: { type: 'disconnect' }
}));
