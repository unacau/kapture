Manual Configuration for Other Assistants

For assistants not listed above, you can manually add Kapture to their MCP configuration:

MCP Configuration

{
  "mcpServers": {
    "kapture": {
      "command": "npx",
      "args": ["-y", "kapture-mcp-server", "bridge"]
    }
  }
}

Common Configuration Locations:

  • VS Code-based editors: Look for settings.json and add under "mcp.servers"
  • JetBrains IDEs: Check plugin settings for MCP configuration
  • Custom integrations: Refer to your tool's MCP documentation

Note: After adding the configuration, restart your assistant to connect to Kapture.
